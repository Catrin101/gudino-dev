# Gudiño Dev — Sitio Web Corporativo

Sitio web oficial de **Gudiño Dev**, empresa de desarrollo de software con sede en Mexicali, Baja California, México.

🌐 **URL:** https://catrin101.github.io/gudino-dev

---

## Sobre el Sitio

Sitio web corporativo bilingüe (Español / Inglés) que presenta los servicios, proyectos y propuesta de valor de Gudiño Dev. Diseñado para atraer clientes B2B que necesitan soluciones de software a medida.

## Stack Técnico

- **HTML5** semántico
- **CSS3** con Custom Properties
- **Vanilla JavaScript** ES6+
- Sin frameworks, sin dependencias de npm
- Selector de idioma integrado (ES / EN) sin recarga
- Animaciones con IntersectionObserver
- Desplegado en **GitHub Pages**

## Estructura

```
gudino-dev/
├── index.html
├── assets/
│   ├── css/        ← Estilos modulares
│   ├── js/
│   │   ├── data/       ← Contenido y traducciones
│   │   ├── components/ ← Componentes renderizados por JS
│   │   └── utils/      ← i18n, animaciones
│   └── img/
└── docs/           ← Documentación de desarrollo
```
